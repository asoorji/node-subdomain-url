# node-subdomain-URL
Retrieve specific posts with a subdomain

The host file is opened on Mac terminal using the command
"sudo nano /etc/hosts" 

sub1 and sub2 are the added subdomains

The payload {
  "id": 2,
  "name": "NAMSN Election",
  "numVoters": 5000
}
can be retrieved using the URL "sub2.localhost:3000"
